
# 📡 WiFi Chat - Offline Nearby Messaging App

WiFi Chat is an Android application that enables nearby devices to communicate without using the internet. The app uses **WiFi Direct (Peer-to-Peer)** to discover nearby devices and establish a direct connection between them. Once connected, users can exchange messages in real time using **socket communication**.

This project demonstrates **Android networking, peer-to-peer communication, and socket programming**.

---

## 🚀 Features

- 📶 Nearby device discovery using **WiFi Direct**
- 🔗 Peer-to-peer connection between Android devices
- 💬 Real-time messaging without internet
- ⚡ Fast communication using **Socket Programming**
- 👥 Automatic role assignment (**Group Owner / Client**)

---

## 🛠 Tech Stack

- **Language:** Kotlin  
- **Platform:** Android  
- **Networking:** WiFi Direct (P2P)  
- **Communication:** Java/Kotlin Sockets  
- **IDE:** Android Studio  

---

## ⚙️ How It Works

1. The app scans for nearby devices using **WiFi Direct discovery**.
2. A connection request is sent to another device.
3. One device becomes the **Group Owner (server)** and the other acts as the **Client**.
4. A **socket connection** is created between the devices.
5. Messages are transmitted instantly through the socket channel.
